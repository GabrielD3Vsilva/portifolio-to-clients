Sobre mim

Olá! Meu nome é Gabriel. Nasci em Osasco, mas atualmente moro em Taubaté. Sou um desenvolvedor web desde 2022. Minha jornada no mundo da programação começou com HTML, CSS e JavaScript. Com o tempo, aprimorei minhas habilidades e explorei outras tecnologias. Tailwind CSS se tornou meu aliado para criar interfaces elegantes e responsivas. Node.js me permitiu desenvolver aplicativos do lado do servidor, enquanto Bootstrap facilitou a criação de layouts consistentes. Além disso, mergulhei no mundo dos bancos de dados. MongoDB me atraiu com sua flexibilidade e escalabilidade, enquanto MySQL me proporcionou uma base sólida para sistemas relacionais.
